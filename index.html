القوالب الجاهزة

حوّل نصك أو ملفاتك إلى بوربوينت

PDF: كل صفحة = شريحة (أو تقسيم ذكي). DOCX/TXT: تقسيم تلقائي أو بفواصل ///.

لو فيه assets/[topic]/bg.jpg أو bg.png هياخدها تلقائي.

إرشادات

  • PDF: اختر “كل صفحة = شريحة” أو “تقسيم ذكي”.
  • DOCX: هنستخرج النص ونقسم تلقائيًا أو بفواصل ///.
  • الخلفية: لو فيه ملف bg.jpg/png داخل مجلد التخصص، هيتطبق تلقائيًا، وإلا نستخدم القائمة.
  • الصور محلية فقط: ضيفها في assets/[topic]/1.jpg … وسيتم الاختيار عشوائيًا لكل شريحة.
